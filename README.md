# Last Horizon

A roguelite shoot-'em-up that fuses Space Invaders-style descending formation pressure, Blazing Lazers-style weapon loadouts, and BALL x PIT-style mid-run recombination.

**Status:** pre-production. Design exploration in progress; no engine or framework selected yet.

## Repository layout

```
docs/
  research/      — external research and references
  design/        — game design document and design notes
  decisions/     — architecture / tech decision records
src/             — game source code (engine TBD)
assets/          — art, audio, fonts, source files for content
tests/           — automated tests
```

See [`docs/README.md`](docs/README.md) for the documentation index.
