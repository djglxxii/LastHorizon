# Assets

## Sprites

v1 placeholder sprites are authored directly for this prototype unless otherwise noted.

- `sprites/player/player-ship.png` — project-authored pixel-art placeholder generated by `tools/generate_player_sprite.gd`.
- `sprites/enemies/baseline-grunt.png` — project-authored pixel-art placeholder generated by `tools/generate_baseline_enemy_sprite.gd`.
- `sprites/enemies/weapon-chip-carrier.png` — project-authored pixel-art placeholder generated by `tools/generate_weapon_chip_carrier_sprite.gd`.
- `sprites/carriers/fuel-cell-carrier.png` — project-authored pixel-art placeholder generated by `tools/generate_fuel_cell_carrier_sprite.gd`.
- `sprites/pickups/weapon-chip.png` — project-authored pixel-art placeholder generated by `tools/generate_weapon_chip_sprite.gd`.
- `sprites/projectiles/pea-bullet.png` — project-authored pixel-art pea-shooter projectile generated by `tools/generate_pea_bullet_sprite.gd`.
- `sprites/projectiles/typed-plasma-bolt.png` — project-authored pixel-art placeholder typed-weapon projectile generated by `tools/generate_typed_projectile_sprite.gd`.

No third-party sprite assets are currently used.
